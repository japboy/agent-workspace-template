# agent-workspace-template

## Constraints

- Use the `agent-memory` skill to log your work and reasoning. Refer back to those logs when restoring context or as needed.
- Consider creating an Agent Skill for reusable qualitative workflows that are not easy to script but can be reliably performed by an agent
- Continuously maintain repository-level Agent Skills using the SkillOpt-informed review principles defined by the `agent-skill-authoring` skill
- Report to the user in Japanese
